# aminul-tech

Personal website for **Aminul Islam** — a full-stack developer and tech enthusiast.

## Features

- **Hero section** with typewriter effect and social links
- **About** section with personal stats
- **Skills** section with animated progress bars
- **Projects** showcase with gradient cards
- **Contact** form with client-side validation
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll, scroll-reveal animations, and sticky navbar

## Tech Stack

Pure **HTML5**, **CSS3**, and vanilla **JavaScript** — zero dependencies, zero build step.

## Running Locally

Simply open `index.html` in your browser, or serve with any static file server:

```bash
npx serve .
```
